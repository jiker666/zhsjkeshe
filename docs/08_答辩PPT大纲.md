# 答辩 PPT 大纲

## 第 1 页：项目标题

- 基于 AI Agent 的自主安全测试可视化平台设计与实现
- 软件工程综合实践课程设计
- 技术栈：Spring Boot + Vue 3 + MySQL

## 第 2 页：项目背景

- Web 安全测试流程分散
- 人工记录日志和报告效率低
- 课程设计需要完整展示软件工程流程

## 第 3 页：需求分析

- 用户登录与权限控制
- 测试任务管理
- Agent 自动生成计划和模拟执行
- 漏洞结果、日志和报告展示
- 首页安全态势可视化

## 第 4 页：系统架构

- 前后端分离
- Vue 3 前端负责交互和可视化
- Spring Boot 后端负责业务和 Agent 流程
- MySQL 存储核心数据

## 第 5 页：功能模块

- 首页仪表盘
- 测试任务
- 漏洞结果
- 测试日志
- 报告中心
- 系统说明

## 第 6 页：Agent 工作流程

- 任务理解
- 计划生成
- 自动执行
- 结果分析
- 报告生成
- 当前为规则模拟版，后续可接入真实大模型

## 第 7 页：数据库设计

- user 用户表
- test_task 测试任务表
- test_result 测试结果表
- test_log 测试日志表
- test_report 测试报告表

## 第 8 页：系统实现展示

- 登录页
- 安全态势仪表盘
- 新建任务示例填充
- 任务详情状态流转
- 漏洞详情弹窗
- 报告预览与导出

## 第 9 页：测试结果

- 后端 `mvn test` 通过
- 前端 `npm run build` 通过
- 主要功能测试用例通过
- 初始化数据支持离线演示

## 第 10 页：项目亮点

- 完整软件工程流程
- Agent 模块可扩展
- 深色安全运营风格界面
- 图表丰富，演示顺畅
- 报告可预览和导出

## 第 11 页：安全合规说明

- 默认面向本地靶场和已授权目标
- 第一版不执行破坏性扫描
- 规则模拟结果用于课程设计展示

## 第 12 页：总结与展望

- 已完成可运行、可演示、可答辩系统
- 后续接入真实 AI Agent
- 增强 RBAC 权限控制
- 增加 PDF/Word 报告导出
- 接入低频授权扫描器和更完整测试规则库
