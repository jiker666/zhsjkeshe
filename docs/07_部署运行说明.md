# 部署运行说明

## 1. 环境准备

- JDK 17+
- Maven 3.x
- Node.js 18+
- MySQL 8.x

## 2. 初始化数据库

在项目根目录执行：

```bash
mysql -uroot -p < sql/init.sql
```

脚本会创建数据库、核心表和演示数据。

## 3. 修改数据库配置

如本机 MySQL 账号密码不是 `root/root`，请修改：

```text
backend/src/main/resources/application.yml
```

重点修改：

```yaml
spring:
  datasource:
    username: root
    password: root
```

## 4. 启动后端

```bash
cd backend
mvn spring-boot:run
```

后端默认端口：`8080`。

## 5. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端默认端口：`5173`。

## 6. 演示账号

```text
admin / 123456
```

## 7. 常见问题

- 数据库连接失败：检查 MySQL 是否启动、数据库密码是否正确。
- 前端接口 401：重新登录，确认 token 是否过期或后端是否重启。
- 报告为空：选择已完成任务，或执行任务后再查看报告。
- 前端构建提示包体积较大：ECharts 和 Element Plus 引起的提示，不影响演示。
