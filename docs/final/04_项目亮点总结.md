# 项目亮点总结

## 1. 软件工程流程完整

项目包含可行性分析、需求分析、总体设计、详细设计、数据库设计、系统实现、软件测试和部署说明。

## 2. 前后端分离

前端使用 Vue 3 和 Element Plus，后端使用 Spring Boot 和 MyBatis Plus，接口清晰，便于扩展。

## 3. 可视化展示充分

首页包含统计卡片和多类图表，漏洞、日志、报告也都有独立页面展示。

## 4. Agent 流程抽象清晰

Agent 被拆分为计划生成、执行、结果分析和报告生成，后续可替换为真实 AI Agent。

## 5. 报告导出能力完整

系统支持 Markdown 预览、HTML 导出、Word 导出和浏览器打印 PDF。

## 6. 演示数据完整

初始化脚本包含用户、任务、漏洞、日志和报告，即使不现场执行任务，也能展示完整效果。

## 7. 易于部署和验收

提供一键启动脚本、环境检查脚本、故障排查文档和教师验收说明。

## 双端与权限亮点

本轮新增管理员端和普通用户端，形成更完整的软件工程系统形态。系统不仅有安全测试任务闭环，还具备注册登录、用户管理、个人中心、角色菜单、路由守卫和后端资源归属校验，能够更好支撑课程设计答辩中的权限设计与越权拦截演示。

## 创新加分项亮点

- Agent 可解释：通过决策链展示任务理解、计划生成、工具选择、测试执行、结果分析和报告生成过程。
- 风险可量化：漏洞结果提供风险评分和风险向量，便于确定修复优先级。
- 漏洞可闭环：整改工单覆盖发现、确认、修复、复测和关闭流程。
- 知识可沉淀：安全知识库保存漏洞成因、影响和修复建议。
- 操作可审计：管理员可追溯登录、任务、漏洞、报告和工单关键操作。
- 演示可稳定：内置演示靶场保证答辩时不依赖外部系统。

## 趣味靶场与开源亮点

- 趣味靶场：通过 6 道本地模拟安全题增强互动性和安全教育属性。
- 动画反馈：答对后显示彩带、积分和成就徽章，演示效果更直观。
- 题库管理：管理员可维护题目，普通用户刷题和查看成绩，双端职责更完整。
- Agent 联动：题目详情提供 Agent 分析思路，但不直接泄露答案。
- GitHub 开源规范：补充 `.gitignore`、`.env.example`、LICENSE、CHANGELOG、SECURITY、CONTRIBUTING 和 CI 工作流。
