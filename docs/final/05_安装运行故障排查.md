# 安装运行故障排查

## 1. MySQL 连接失败

检查 MySQL 是否启动，账号密码是否正确。可通过环境变量修改：

```bash
export DB_USER=root
export DB_PASSWORD=你的密码
```

macOS Homebrew 可尝试：

```bash
brew services start mysql
```

## 2. 8080 端口占用

脚本会自动切换到 18080。手动启动可使用：

```bash
SERVER_PORT=18080 mvn spring-boot:run
```

## 3. 5173 端口占用

脚本会自动切换到 5174。手动启动可使用：

```bash
npm run dev -- --port 5174
```

## 4. npm install 失败

检查 Node.js 和 npm 版本，建议 Node.js 18+。如网络慢，可切换 npm 镜像。

## 5. Maven 下载慢

首次运行会下载 Spring Boot、MyBatis Plus、Apache POI 等依赖。网络慢时可以配置 Maven 镜像。

## 6. Java 版本不对

项目要求 Java 17+。可执行：

```bash
java -version
```

## 7. 前端请求后端失败

检查 `frontend/.env.development` 中的 `VITE_API_BASE_URL` 是否指向实际后端端口。

## 8. 登录失败

确认数据库已初始化，并使用默认账号：

```text
admin / 123456
```
