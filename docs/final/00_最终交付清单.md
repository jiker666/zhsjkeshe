- `backend/`：Spring Boot 后端源码，包含认证、任务、Agent、漏洞、日志、报告、健康检查等模块。

## 8. 主要验收点

- 能登录系统。
- 能查看首页安全态势。
- 能新建和编辑测试任务。
- 能执行 Agent 模拟测试。
- 能查看漏洞详情和修改处理状态。
- 能查看日志时间线。
- 能预览报告并导出 HTML、Word 或打印 PDF。
- 能查看 Agent 决策链和风险评分。
- 能从漏洞生成整改工单并执行模拟复测。
- 能查看安全知识库和管理员操作审计。
- 能使用内置演示靶场创建稳定演示任务。
