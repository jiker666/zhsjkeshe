# Security Policy

## 安全声明

本项目是课程设计和安全教育演示系统，仅用于本地靶场、自建测试系统、课程演示数据或已获得明确授权的环境。

请不要将本项目用于未授权目标，不要对公网系统进行扫描、爆破、绕过认证或破坏性测试。

## 使用边界

- 内置演示靶场接口只返回本地模拟数据。
- 规则模拟 Agent 不执行真实攻击扫描。
- 弱口令、越权、信息泄露等内容均为课程设计演示场景。
- 使用者应遵守当地法律法规和所在学校/组织的安全规范。

## 报告项目自身安全问题

如果你发现本项目代码本身存在安全问题，请通过 GitHub Issue 或课程项目维护者联系方式反馈。请不要公开泄露可被滥用的细节，建议先给出影响范围和修复建议。
